# Perguntas novas para revisão — Rodada 5 (unificação de catálogo com o Nexo Coger)

Data: 2026-07-10

Lista de todas as 206 perguntas criadas na Rodada 5 para as 41 normas do catálogo unificado que antes não tinham banco de perguntas próprio no Oitiva 360, agrupadas por norma na ordem de entrega (seção 3, prioridade Grupo F → D → G → C → A → B → E). Todas estão marcadas com `revisao: "pendente"` no catálogo — este documento serve de checklist para a revisão de conteúdo jurídico, conforme a seção 3.6 e o critério de aceite nº 7 do documento da Rodada 5.

Cada pergunta é identificada por `id`, bloco (finalidade no roteiro) e o texto exato como aparece no catálogo.
